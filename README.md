# 🩺 GastroGuard – Lifestyle-Based Gastritis Risk Prediction

A Machine Learning-based web application developed as part of my **AI & Data Analytics Internship** to predict the risk of gastritis based on a patient's health, clinical, and lifestyle information.

---

## 📌 Project Overview

GastroGuard analyzes patient information using a trained **Logistic Regression** model and predicts the likelihood of gastritis. The application provides an estimated risk percentage, risk category, personalized health recommendations, and stores patient records using Google Sheets integration.

---

## ✨ Features

- 🔐 User Login System
- 📋 Health Assessment Form (36 Features)
- 🤖 Machine Learning-Based Risk Prediction
- 📊 Risk Percentage Gauge
- 💡 Personalized Health Recommendations
- 📁 Patient Records Dashboard
- ☁️ Google Sheets Integration
- 📄 Export Patient Records to CSV

- Lifestyle Factors
- Medication History
- Bowel Health
